# scottbertrand.com — V1.5

## Purpose
This repository contains V1.5 of scottbertrand.com.

V1.5 is a **conversion-first Brand & Web Systems consultancy site** designed to:
- Capture inbound leads in **Greater Sudbury / Northern Ontario**
- Support **remote, Canada-wide clients**
- Drive **paid audit inquiries** as the primary conversion

This is not a portfolio site or content hub.

---

> Based in Greater Sudbury, serving Northern Ontario — available remotely across Canada.

Must include:

  “Brand & Web Systems for Greater Sudbury & Northern Ontario”
