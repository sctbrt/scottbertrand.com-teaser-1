What's Included

Brand Positioning

Value proposition development, target audience definition, competitive differentiation, and brand personality framework.

Visual Identity System

Logo design (1 primary concept + 1 revision round), color palette, typography system, and supporting graphic elements.

Brand Guidelines

Comprehensive PDF document covering logo usage, color system, typography, imagery style, voice & tone, and application examples.

Asset Package

Logo files in multiple formats (SVG, PNG, EPS), brand color values (HEX, RGB, CMYK), and typography recommendations.

Collaborative Process

Initial discovery session, mid-project check-in, and final presentation with handoff materials and implementation guidance.

2 Revision Rounds

One round after initial concepts, one round after refined direction. Keeps project on track while ensuring you're satisfied.

How It Works

1

Discovery (Week 1)

Common Questions

What if I need more than 2 revision rounds?

Additional revisions are available at $200/round. However, the structured process is designed to get to the right solution within 2 rounds.

Can I add a website to this?

Yes. You can bundle this with Website Foundation for the full package price ($6,900 instead of $7,250 separately).

Do I own all the files?

Yes. Upon final payment, you own all brand assets and can use them however you like.

What file formats do I receive?

Logo files in SVG, PNG (transparent), EPS, and high-res JPG. Brand guidelines as PDF. Source files available upon request.

Can you help implement the brand?

The service includes implementation guidance in the brand guidelines. For hands-on implementation (website, marketing materials, etc.), we can discuss additional services.
