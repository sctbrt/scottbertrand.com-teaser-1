This Is A Good Fit If...

Good Fit

  • You need a new website built from scratch
  • You want a fast, modern site without WordPress complexity
  • Your site needs 7 pages or fewer (e.g., Home, About, Services, Contact)
  • You value performance, accessibility, and clean code
  • You're ready to provide content (or we can discuss copywriting separately)

Not A Good Fit

  • You need a complex web application or e-commerce site
  • You need more than 7 pages (contact for custom quote)
  • You want a WordPress site with a page builder
  • You need content management for frequent updates (consider a CMS)
  • You need ongoing development work (available separately)

Common Questions

Is there a call involved?

No calls required—this is delivered async with written feedback rounds. If clarification would help, you can add a Phone Consultation ($125) at any point.

What if I need more than 7 pages?

Additional pages are $300 each. For larger sites (15+ pages), contact me for custom pricing.

Can you write the content for me?

I provide content strategy and outlines, but you provide the actual copy. Professional copywriting is available as an add-on for $150/page.

Do you build e-commerce sites?

Not as part of this package. For e-commerce, I recommend platforms like Shopify or contacting me for a custom quote.

What if I don't have a brand yet?

I can create a simple visual identity as part of the website, or you can bundle with Brand Reset for the full package price. See Brand + Website Reset.
